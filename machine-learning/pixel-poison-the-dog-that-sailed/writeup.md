# Pixel Poison: The Dog that Sailed — Machine Learning | 500 pts

## Description

> A dog sailing the ocean... magic? No. Mathematics. Fool our AI with changes invisible to the human eye.
>
> The `cliente.py` file contains the code to send two images to the server. The `attack.py` file is the one you must modify to fool the AI and make it believe a dog is a ship.
>
> A couple of papers that may be useful:
> - https://arxiv.org/abs/1412.6572
> - https://arxiv.org/abs/1706.06083
>
> http://172.210.57.123:8888

## Files

- `cliente.py` — client code to send images to the server
- `attack.py` — file to modify to perform the adversarial attack
- `dog_sample.png` — sample dog image to fool the classifier
- `model.pth` — the target model
