# Corrupted Image — Miscellaneous | 200 pts | Easy

## Description

> We have recovered a file that should be an important photograph, but it is corrupted and cannot be opened with any image viewer. We believe the corruption is at the very beginning of the file. Your mission is to inspect the file at the binary level, fix the anomaly, and recover the image that contains the flag.

> **Hint 2 (free):** Look up the official signature for the file extension of this challenge. This standard opening byte sequence is easy to find online.
