# Mensajes Gemelos — Crypto | 400 pts | Easy

## Description

> A spy has intercepted two encrypted messages that were sent using the same encryption method and, suspiciously, the same secret key. Analysts believe that one of the messages contains known public information, while the other contains the flag you are looking for.
>
> Can you exploit this cryptographic vulnerability to recover the secret message?

## Hints

> **Hint 1 (free):** When two messages are encrypted with XOR using the same key, something interesting happens:
>
> If C1 = M1 ⊕ K and C2 = M2 ⊕ K, then: C1 ⊕ C2 = (M1 ⊕ K) ⊕ (M2 ⊕ K) = M1 ⊕ M2

> **Hint 2 (free):** You know the first message starts with "Hola mundo desde la universidad". The cipher used is XOR. What can you obtain by following the property shown in the first hint?

> **Hint 3 (50 pts):** Not used.
