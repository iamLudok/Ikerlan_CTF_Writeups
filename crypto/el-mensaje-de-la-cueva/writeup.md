# El Mensaje de la Cueva — Crypto | 200 pts | Easy

## Description

> A mysterious message has appeared on the wall of a cave. It seems to be in an ancient language, but a clue left by an explorer tells you that you should not read the message, but observe it carefully. The first digit of the lock is hidden in a very subtle clue.

## Hints

> **Hint 1 (free):** Listen carefully to the cadence and rhythm. The message is composed of the repetition of only two types of sounds: a short one ('dit') and a long one ('dah').

> **Hint 2 (free):** It is a telegraphic communication system that helped connect the world in the 19th century. Look for the alphabet that uses dots and dashes for each character.

> **Hint 3 (50 pts):** Not used.
