# El Silencio del Retraso — Hardware | 500 pts | Medium

## Description

> The cybernetic syndicate "Los Cronómetros" has launched a new ultra-secure authentication program to protect their most valuable database: the Secret Numbers Index (INS), a list of digit codes that control their illicit financial network.
>
> They claim their C code is unbreakable thanks to rigorous optimisations and an opaque structure designed to confuse any analyst. They have promised that the code is so complex that any attempt at static analysis will result in hours of frustration.
>
> Your mission is to recover the master PIN code (the flag). You don't have access to the database — only to the small executable that performs password validation.
>
> The number of digits is unknown, but it is a numeric key.
