# GPIO Binary Password — Hardware | 100 pts | Easy

## Description

> You found a forgotten Arduino in the robotics classroom. It is powered on and all its digital pins are in different states (HIGH or LOW). A post-it stuck to the Arduino reads:
>
> *"If you read the pin states correctly, you will discover the lab password.*
>
> *Pins D2-D9 contain the secret. HIGH = 1, LOW = 0"*
>
> The flag format is `ikerlan{contraseña}`

## Hints

> **Hint 1 (free):** Arduino digital pins can be in two states: HIGH (5V, logic 1) or LOW (0V, logic 0). When you read 8 pins together, you have 8 bits — exactly what you need to represent an ASCII character. There are 13 different readings because the password has 13 characters.

> **Hint 2 (free):** The password is a 13-character phrase that includes uppercase letters, lowercase letters, underscores, and digits.
